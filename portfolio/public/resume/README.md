# Resume Files

This directory contains downloadable resume files in different formats:

- `Malith_Rajamanthri_Resume.pdf` - PDF version of the resume
- `Malith_Rajamanthri_Resume.docx` - Word document version of the resume

## Note for Development

In a real deployment, you would place the actual resume files here. For development purposes, the download functionality will attempt to download these files, but they may not exist yet.

To add your resume files:

1. Create your resume in PDF format and save it as `Malith_Rajamanthri_Resume.pdf`
2. Create a Word document version and save it as `Malith_Rajamanthri_Resume.docx`
3. Place both files in this directory

The resume download component will automatically serve these files when users click the download buttons.
